## 리액트 3주차 스타일 컴포넌트 과제 완성

```bash
📂 src
┣  🗂 App.jsx : Buttons, Input, Modal, Select 컴포넌트를 보여주는 메인페이지 App.jsx
┣ 📂 components
┃   ┣  🗂 Button.Style.jsx : primary, negative 버튼 컴포넌트
┃   ┣  🗂 StyleCollection.jsx : 스타일 속성 모음 컴포넌트
┣ 📂 hooks
┃   ┣ 🗂 useModal.jsx : features/Modal 에 쓰이는 커스텀 훅(함수 기능, 버튼에 따른 모달영역 컴포넌트 포함)
┃   ┗ 🗂 useSelect.jsx : features/Select 에 쓰이는 커스텀 훅(함수 기능, 셀렉트 버튼에 따른 셀렉트박스 컴포넌트)
┗ 📂 features
   ┣ 🗂 Buttons.jsx : 버튼영역 컴포넌트
   ┣ 🗂 Input.jsx : 인풋영역 컴포넌트
   ┣ 🗂 Modal.jsx : 모달영역 컴포넌트
   ┗ 🗂 Select.jsx : 셀렉트영역 컴포넌트
```
